Finding Pointers With Cheat Engine

Friday, February 16, 2024
5 min read

1. Installing Cheat Engine

Cheat Engine can be found at the following website: https://cheatengine.org/, to download, click the big green "Download Cheat Engine" button.

2. Setting Up

Run Cheat Engine and Click the Rainbow icon in the top left, once clicked the Process List will show up, select the game you are using, for this i will be using Assualt Cube.

3. Finding the value

Choose a value you would like to modify, I will be choosing the ammo. In the top right find the "value:" box and enter the value of the ammo, health, etc that you are lookign for e.g 20, once it is entered click the "First Scan" button above, once it has finished scanning do something in-game to change the value e.g shooting the gun if you are finding the ammo, once the value has changed next to the "First Scan" button click "Next Scan" once it has finished scanning repeat these steps (the steps of changing the value and scanning). Once there a few addresses left change them and see which one changes the value in-game.

4. Finding the Pointer

Right cick the address and click "Pointer Scan For This Address" -> "Ok" -> save the file, once you have done this close the game and KEEP Cheat Engine OPEN!!!

Open the game and attach it to Cheat Engine again and find the address of the value again, once done go back to the pointer scanner and click the "Pointer Scanner" button.

Click the "Rescan Memory" button and type in the new address you found and click "OK".
